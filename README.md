# 🚀 Aryan Singh — Personal Portfolio

> A fully responsive, modern personal portfolio built with React and Tailwind CSS — showcasing projects, skills, and contact information.

🔗 **Live Demo:** [aryan-singh-portfolio-rouge.vercel.app](https://aryan-singh-portfolio-rouge.vercel.app/)

---

## 📌 Overview

This is my personal developer portfolio — a clean, fast, single-page application designed to present my work, technical skills, and background as a web developer. Built with React and styled using Tailwind CSS, it's crafted to leave a strong first impression while being easy to navigate across all devices.

---

## ✨ Features

- 👤 **About Me Section** — A personal introduction and background as a web developer
- 🛠️ **Skills Showcase** — A visual display of technical skills and tools
- 💼 **Projects Gallery** — Highlighted projects with descriptions and live/repo links
- 📬 **Contact Section** — Easy ways to get in touch or connect
- 📱 **Fully Responsive** — Seamless experience across desktop, tablet, and mobile
- ⚡ **Fast & Lightweight** — Optimized performance with minimal load times
- 🎨 **Modern Design** — Clean UI with smooth layout powered by Tailwind CSS

## 🚀 Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/your-username/portfolio.git
   cd portfolio
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm start
   ```

   The app will run at `http://localhost:3000`

## 🌐 Deployment

This portfolio is deployed on **Vercel**. To deploy your own version:

```bash
npm run build
```

Connect your GitHub repository to [Vercel](https://vercel.com/) for automatic deployments on every push to `main`.

## 🎨 Customization

Want to fork and make it your own? Here's what to update:

- **`src/components/Hero`** — Your name, title, and tagline
- **`src/components/About`** — Your personal bio and background
- **`src/components/Skills`** — Your tech stack and tools
- **`src/components/Projects`** — Your project details, links, and screenshots
- **`src/components/Contact`** — Your email and social media links
- **`src/assets/images`** — Your profile photo and project thumbnails
